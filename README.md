# homebrew-inkess

Homebrew Cask for [Inkess](https://github.com/gezhigang000/inkess-app) — Markdown Reader & Converter.

## Install

```bash
brew tap gezhigang000/inkess
brew install --cask inkess
```

## Update

```bash
brew upgrade --cask inkess
```

## Uninstall

```bash
brew uninstall --cask inkess
```
